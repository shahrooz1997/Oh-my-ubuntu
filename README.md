# Oh-my-ubuntu
A bash script to bootstrap your fresh-installed Ubuntu

Run the following command:
`sudo apt-get install -y git && git clone https://github.com/shahrooz1997/Oh-my-ubuntu.git && cd Oh-my-ubuntu && ./install.sh`
